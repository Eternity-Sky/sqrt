## 什么是 sqrt？

**sqrt** 是一个基于 GitHub Pages、React 和 CP OAuth 的先进博客系统框架。它具有以下特点：

- **零服务器**：完全运行在 GitHub Pages 上，不需要自己的服务器。
- **先进架构**：使用 Vite, React 和 Tailwind CSS 构建。
- **在线发表**：集成 GitHub API，支持直接在网页端撰写并发表 Markdown 文章。
- **安全登录**：使用 CP OAuth 提供安全的用户认证。
- **现代 UI**：基于 Framer Motion 的优雅动画和极致的阅读体验。
